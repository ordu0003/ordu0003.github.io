LEARNING HTML

Tags I've learned:

Image:
This tag allows displaying an image on a website. We can use an 'alt' attribute to describe it.

Anchores:
The tag lets us link websites, documents, images, and more from our website to another page.

Headings:
We can add a heading with a different hierarchy (h1, h2, h3, h4...) and organize the content better.

Important text:
The text can be emphasized by its weight with this tag. The tag makes the word bolder than the other text.

Blockquote:
To create a large quote separated from the text. It can be used together with a 'cite' tag to add the author of the quote.
